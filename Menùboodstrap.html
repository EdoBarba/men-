HTML

HTML è un linguaggio che permette di impaginare e formattare pagine collegate fra di loro attraverso link - ovvero, i siti web.

CSS

CSS, o Cascading Style Sheets, è un linguaggio usato per programmare la resa grafica di documenti scritti in un linguaggio di markup, come HTML e XML..

C++

C++ è un linguaggio di programmazione a oggetti tra i più diffusi in svariati ambiti dell'informatica professionale, da quello dello sviluppo mobile agli ambienti scientifici, passando per il gaming e lo sviluppo di GUI.

LOGICA

per programmare con un linguaggio di programmazione bisogna conoscerlo per poi saperlo sfruttare

JAVASCRIPT

JavaScript è un linguaggio di scripting lato client utilizzato per rendere interattive le pagine web. JavaScript, insieme ad HTML e CSS, è una delle tecnologie principali della programmazione web front end

SISTEMI

il sistema indica in modo generico un computer o un insieme di più computer o altri apparati elettronici (es. router ecc...), tra loro interconnessi in rete: si compone sia di elementi hardware che software.

JAVA

Java è un linguaggio di programmazione generico utilizzato per lo sviluppo web. Java è uno dei linguaggi di programmazione più popolari in uso, in particolare per le applicazioni web client-server.
